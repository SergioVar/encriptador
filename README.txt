Proyecto de Encriptación y Desencriptación de Mensajes


Aplicación web para encriptar y desencriptar mensajes, desarrollada con HTML, CSS y JavaScript.
- Encriptar mensajes
- Desencriptar mensajes encriptados previamente
- Reiniciar el contenido del área de texto

Tecnologías Utilizadas
- HTML
- CSS
- JavaScript

Créditos
Este proyecto fue creado con el apoyo de tutoriales de YouTube y herramientas de Alura Latam.

Uso
1. Ingresa el texto que deseas encriptar en el área de texto.
2. Haz clic en "Encriptar" para encriptar el mensaje.
3. El mensaje encriptado aparecerá en la sección correspondiente.
4. Para desencriptar un mensaje, ingresa el texto encriptado y haz clic en "Desencriptar".
5. Usa "Reiniciar" para limpiar el área de texto y comenzar de nuevo.
